\subsection{Subsection title}
\label{sec:subsection-title}

\pptext{Crossreference.} \textbf{USE PACKAGE cleveref} to avoid typos.

Use \verb!\cref{sec:section-title}! to refer to \cref{sec:section-title}.

\Cref{sec:subsection-title}, use \verb!\Cref{sec:subsection-title}! when the reference is at the beginning of a sentence.

Use starred variants (e.g. \verb!\cref*{sec:section-title}!) to avoid creating hyperlinks. For example, \cref*{sec:section-title}.
\medskip

\pptext{Code.} Use \verb!\cd{code}! to use the inconsolata font as: \cd{code}.
\medskip

\pptext{Diamond.} Use \verb!\diam! to add a diamond symbol at the end of a line: (used for examples sections) \diam
\medskip
